# Task 1: Basic Network Scanning with Nmap

## Objective
Perform a network scan using Nmap to identify open ports and services on the system.

## Tool Used
- Nmap

## Steps Performed
1. Captured system info using `ipconfig`.
2. Opened Nmap and performed a TCP SYN scan on localhost.
3. Captured screenshots of Nmap output.

## Screenshots
- `nmap_version.png` → Nmap version
- `ipconfig.png` → IP address
- `scan_running.png` → Scan in progress
- `scan_result.png` → Final scan result
